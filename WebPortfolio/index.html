프로젝트 설명

“강아지는 훌륭하다”는 강아지를 반려동물로 둔 견주들을 위한 사이트입니다. 실시간 동물 관련 뉴스를 제공하며, 견종백과를 통해 다양한 정보를 제공합니다. 또한, 카카오맵을 통해 강아지 유치원이 있는 곳을 각 지역별 위치 정보로 제공하며, 게시판에서 여러 견주님들끼리 다양한 의견을 주고받을 수 있습니다. 마지막으로, 해당 사이트에서 추천하는 강아지 관련 Youtube 영상을 시청할 수 있습니다.

적용기술

- 프레임워크 : Spring Framework
- 데이터베이스 : MySQL, MongoDB
- 적용 데이터 : bugs 뮤직 차트
- 기타 : komoran형태소분석기, JS, Jsoup, jQuery, Ajax, HTML5, Bootstrap
- 클라우드 : AWS EC2

적용기술

- 프레임워크 : Spring Framework
- 데이터베이스 : Oracle 11g
