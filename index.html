#LAP #노래 가사 분석

프로젝트 설명

"LAP(Lyric Analysis Page)"는 노래순위차트의 노래가사들을 분석하는 웹페이지 형식의 서비스 입니다. Bugs노래순위차트를 실시간으로 제공하며 순위 100위에 노래가사들의 긍정적인 수치를 각도 차트로 보여줍니다. 또한, 노래가사에서 제일 많이 쓰는 단어를 워드클라우드로 보기 쉽게 보여줍니다. 게시판에서 노래에 대한 다양한 의견을 주고받을 수 있습니다.

적용기술

- 프레임워크 : Spring Framework
- 데이터베이스 : MySQL, MongoDB
- 적용 데이터 : bugs 뮤직 차트
- 기타 : komoran형태소분석기, JS, Jsoup, jQuery, Ajax, HTML5, Bootstrap
- 클라우드 : AWS EC2

#Animal Hospital #동물병원을 찾아라

프로젝트 설명

"Animal Hospital"는 동물병원의 위치 및 건의사항 수집 혹은 정보제공을 위한 웹페이지입니다.

적용기술

- 프레임워크 : Spring Framework
- 데이터베이스 : Oracle 11g
- 적용 공공데이터(OpenAPI) : 서울열린데이터광장, 경기도 데이터 드림 동물병원 정보 OpenAPI
- 기타 : KakaoMap API, summernote, JS, jQuery, HTML5, Bootstrap
